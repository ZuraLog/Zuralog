Live Session — What it does Coming Soon

  • Running timer — counts up from 0:00 the moment you start. Persists in the background if you close the app.
  • Exercise library — search from 5,000+ exercises with muscle group filters. Add them one by one as you work through your session.
  • Set-by-set logging — tap each set as you complete it. Log reps and weight inline. Previous set's values pre-fill the next set.
  • Rest timer — automatic countdown between sets. Configurable per exercise.
  • Finish session — saves the full session log, total duration, volume, and exercises. Feeds into the AI and Progress tab.
  • This is a significant standalone feature — similar scope to building a mini Lyfta inside Zuralog. Planned for a future sprint.
Why "Coming Soon" and not hidden? Showing the live session button (greyed out, with a "Coming Soon" badge) does two things: it tells gym users the feature is on the roadmap so they don't feel the app is missing it, and it creates genuine anticipation. When it ships, the mode picker already exists — we just enable the button.
