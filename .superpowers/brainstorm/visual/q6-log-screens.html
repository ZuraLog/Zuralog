Question 6 of several

When you tap a log type in the bottom sheet grid — say "Workout" — what happens next? This affects how fast and how rich the experience feels. There's a trade-off between speed and detail.

A

Inline — sheet expands in place

Tapping a type expands the same bottom sheet to show that type's fields. No new screen. Fast, feels native. Grid animates out, fields animate in. Tap back to return to the grid.

Grid → tap Workout → fields slide in → Submit → done
B

New sheet — each type opens its own bottom sheet

Tapping a type closes the grid and opens a dedicated sheet for that type. Each log type can have a fully custom layout. Slightly more steps but maximum flexibility per type.

Grid sheet closes → Workout sheet opens → Submit → done
C

Full screen — each type gets a dedicated page

Tapping opens a full-screen log entry page. Most room for detail — great for Workout or Run where you want sets, reps, distance, pace. Heavier feel, but richer experience.

Grid → tap Workout → full screen page → Save → back to Today
D

Hybrid — simple types stay in sheet, complex get full screen

Quick things like Water, Mood, Weight stay in an inline sheet (one tap, one value, done). Complex things like Workout, Run, Meal open a full screen. Best of both worlds.

💧 Water → inline +1 glass → done in 1 tap 🏋️ Workout → full screen with sets/reps/duration 😊 Mood → inline slider → done in 2 taps 🏃 Run → full screen with distance/pace/map
