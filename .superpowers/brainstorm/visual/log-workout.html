Workout — Field Breakdown

  • Workout type — Tap-to-select chips: Weights, Cardio, HIIT, Yoga, Pilates, CrossFit, Class, Other. Required. Single select.
  • Duration — Number input in minutes. Required. If Strava / Apple Health is connected and detected a workout, pre-filled. auto-synced
  • Intensity — 5-dot scale (Light → Max). Quick to fill, feeds the AI. optional
  • Exercises — Each entry has a name, sets, reps, and weight. Expandable list — add as many as you want. Pre-populated from last time you logged the same workout type (e.g. last Weights session). optional
  • How did it feel? — 4-option emoji rating: Rough / OK / Solid / Beast. Simple and fun. Helps the AI track recovery and performance patterns. optional
  • Notes — Free text for anything else. optional
  • Required fields — Only type and duration. A user can log a workout in 3 taps.
Smart pre-fill: When a user opens the Workout log, if they logged the same workout type recently, we pre-fill the exercise list from last time. They just adjust the weights/reps. This removes the biggest friction point for gym users — re-entering the same exercises every session.
