What's on screen, top to bottom

  • App bar — App name, notification bell, profile avatar
  • Greeting + Streak — Time-aware ("Good morning/evening") + flame streak badge
  • Health Score + Log Ring — Side by side. Score = how you're doing. Ring = what you've logged today vs. what's missing.
  • Snapshot Cards — Horizontally scrollable row of today's key numbers: steps, water, sleep, weight, calories, mood
  • Daily Goals — Lightweight progress bars. Water: 4/8 glasses. Steps: 8,200/10,000. Not deep — just today.
  • AI Insights — Scrollable cards. Tap any to read the full analysis. Unread dot on new ones.
  • FAB — Always visible bottom-right. Sage green. Opens the log grid.

Log Grid Sheet

  • Slides up from the bottom when + is tapped
  • 4-column icon grid — all 12 types visible at once
  • Already-logged types today get a small green checkmark so you know what's done
  • Drag handle at top, dismissable by swiping down or tapping outside
  • Simple types inline — Water, Mood, Energy, Stress, Weight, Steps, Calories expand within the sheet
  • Complex types full screen — Workout, Run, Sleep, Meal, Supplements, Symptom open a dedicated page

Simple Log — Inline

  • Grid animates out, fields slide in — same sheet, no navigation
  • Water — +/− counter, shows today's count vs. goal, Save button
  • Mood / Energy / Stress — labeled slider (1–10), colour-coded, Save
  • Weight — number input with unit (kg/lbs), Save
  • Steps / Calories — number input, Save
  • Haptic feedback on Save. Sheet dismisses automatically. Log Ring on Today tab updates instantly.
  • Back arrow returns to the grid without saving

Complex Log — Full Screen

  • Opens as a full screen page from the sheet — feels intentional, not rushed
  • Workout — type chips, duration, intensity, notes, optional sets/reps
  • Run/Cardio — distance, duration, optional pace/route note
  • Sleep — bedtime, wake time, quality slider
  • Meal — meal name/description, optional calorie estimate
  • Supplements/Meds — multi-select pill chips from saved list + add new
  • Symptom — body area chips, severity slider, notes
  • Save button fixed at bottom. Back button discards without saving (with confirmation if fields filled).

12 Log Types via FAB

UX Principles

  • Simple logs done in 1–2 taps
  • Complex logs get room to breathe
  • Log Ring updates instantly after logging
  • Already-logged types show a checkmark in grid
  • All components reusable — UI team can reskin without touching logic
