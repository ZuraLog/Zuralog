Sleep — Field Breakdown

  • Bedtime + Wake time — Time pickers. If Apple Health / Health Connect is connected, pre-filled automatically. auto-synced Duration is calculated and shown instantly ("8h 00m").
  • Sleep quality — 5-face emoji picker (Awful → Great). Maps to 1–5 internally. optional
  • Night interruptions — Simple +/− counter for how many times you woke up. optional
  • Factors — Tap-to-select chips: Late workout, Alcohol, Stress, Caffeine, Screen time, Napped, Noisy, Hot/Cold. These feed the AI's correlation engine — e.g. "Sleep is 23% worse on nights you log alcohol." optional
  • Notes — Free text. optional
  • Required fields — Only bedtime and wake time are required. Everything else is clearly marked optional so the user never feels forced.
Why factors matter: The sleep factors chips are small to fill in but extremely valuable for the AI. When a user consistently tags "Late workout" on bad sleep nights, the AI can surface that as an insight. This is what makes Zuralog smarter than a plain sleep tracker.
